Core Concepts Quick Reference
Key ideas at a glance — expand each for details

📊 Quantitative Data

  • Uses numbers and measurements
  • Examples: 5 cm tall, 23°C, 14 birds
  • Collected with tools: ruler, scale, timer
  • Can be averaged and graphed numerically

📝 Qualitative Data

  • Uses words and descriptions
  • Examples: red flower, sweet smell, rough texture
  • Cannot be directly calculated
  • Organized in tables, not number lines

📈 Bar Graph

  • Compares different categories
  • Bars do not touch (usually)
  • Best for categorical data
  • X-axis: categories, Y-axis: values

📉 Line Graph

  • Shows trends over time
  • Points connected by lines
  • X-axis: time, Y-axis: measured value
  • Can interpolate and extrapolate

🥧 Pie Chart

  • Shows parts of a whole (100%)
  • Each slice = one category
  • Best for proportional data
  • Not good for change over time

🔬 Independent Variable

  • What the scientist changes on purpose
  • Goes on the X-axis
  • Only ONE per controlled experiment
  • Example: amount of fertilizer

📏 Dependent Variable

  • What the scientist measures
  • Depends on the independent variable
  • Goes on the Y-axis
  • Example: plant height in cm

🔒 Controlled Variables

  • Kept the same in all groups
  • Make the test fair
  • Also called constants
  • Example: same soil, water, sunlight

➕ Mean

  • Sum of all values ÷ count
  • Most common average
  • Affected by outliers
  • Example: (4+5+6+7+8)÷5 = 6

⬛ Median

  • Middle value when sorted
  • Not affected by outliers
  • Even count: average two middles
  • Example: 4,5,6,7,8 → median = 6

🔁 Mode

  • Most frequently occurring value
  • Can have 0, 1, or multiple modes
  • Good for categorical data
  • Example: 3,5,5,7,9 → mode = 5

↔️ Range

  • Max value minus min value
  • Shows how spread out data is
  • Example: 4,5,6,7,8 → 8−4 = 4
  • Large range = more variability
Accuracy vs. Precision

🎯 Accuracy

  • How close to the TRUE value
  • Can be accurate but not precise
  • Example: measuring 50g when true = 50g
  • Important for validity of experiment

🎰 Precision

  • How CONSISTENT measurements are to each other
  • Can be precise but not accurate
  • Example: always getting 47g (but true = 50g)
  • Good experiment needs BOTH
Key Vocabulary
Know every term before test day
TermDefinition
DataInformation gathered during an experiment. Can be numbers (quantitative) or descriptions (qualitative).
QualitativeData that uses words or descriptions, not numbers. Example: "The liquid was clear and odorless."
QuantitativeData that uses numbers and measurements. Example: "The plant grew 5.2 cm in one week."
Independent variableThe variable the scientist intentionally changes in an experiment. Plotted on the X-axis.
Dependent variableThe variable that is measured — it depends on the independent variable. Plotted on the Y-axis.
Controlled variableVariables kept the same in all groups to ensure a fair test. Also called constants.
HypothesisA testable prediction: "If [independent variable], then [dependent variable]." Must be falsifiable.
MeanThe average: sum of all values divided by the number of values.
MedianThe middle value when all data points are arranged in numerical order.
ModeThe value that appears most frequently in a data set. A set can have no mode, one mode, or multiple modes.
RangeThe difference between the maximum and minimum values in a data set.
OutlierA data point that is very different from the rest of the values in a data set.
AccuracyHow close a measurement is to the true or accepted value.
PrecisionHow consistent or repeatable measurements are — how close measurements are to each other.
InterpolationEstimating a value between two known data points on a graph.
ExtrapolationEstimating a value beyond the known data range by extending the trend.
ConclusionA statement explaining what the data shows, supported by evidence from the experiment.
TrendA consistent pattern or direction of change visible in data or on a graph.
Bar graphA graph that uses bars to compare different categories of data.
Line graphA graph that connects data points with lines to show change over time.
Pie chartA circular graph divided into sections showing each category as a percentage of the whole.
